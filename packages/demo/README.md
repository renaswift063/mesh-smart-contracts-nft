# Mesh Playground

## How to start the Mesh Playground on localhost

From the root folder...

#### 1. Setup
Run:
```sh
yarn install
```

#### 2. Start module building
Run:
```sh
yarn run start:module
```

#### 3. Start Playground
Open a new Terminal and:
```sh
yarn run start:demo
```
and visit [http://localhost:3000/](http://localhost:3000/)
